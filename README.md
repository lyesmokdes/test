# Team23

Execution :

À partir du répertoire build, exécuter : java -jar Team23.jar

	- Supporte l'éxécution en run ou debug dans Netbeans: utilisation de fichiers par default dans le .jar
	- Supporte lancement par ligue de commande:
		- 1 paramètre  => fichier d'entrée, donc fichier resultat par défaut
		- 2 parametres => fichier d'entrée et de sortie
